Place your favicon assets here. Recommended set:

- `favicon.ico` (multi-size 16/32/48)
- `favicon-32x32.png`
- `favicon-16x16.png`
- `apple-touch-icon.png` (180x180)
- `site.webmanifest`
- `safari-pinned-tab.svg` (if you want a pinned tab icon)

Update `index.html` head links to point to these once added.
